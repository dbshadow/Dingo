1. IDML to CSV Extractor

This tool extracts text from an Adobe InDesign IDML file and converts it into a CSV file ready for translation.

Note: The extracted CSV file (with `source`/`target` columns) is not directly compatible with the 'CSV Translator' tab. You must manually change the headers to BCP-47 language codes (e.g., `en`, `de`, `zh-Hant`) to use it there.

2. Rebuild IDML from CSV

This tool merges a translated CSV file back into the original IDML file structure.

Note: The CSV file used for rebuilding *must* contain `source` and `target` columns. Files downloaded from the 'CSV Translator' tab need their headers modified before they can be used here.
